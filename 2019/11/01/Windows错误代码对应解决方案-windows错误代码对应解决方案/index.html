Windows错误代码对应解决方案

Created2019-11-01|Updated2022-05-28|Other
|Word count:47|Reading time:1min|Post View:

错误代码0x80131500

打开“运行”输入 inetcpl.cpl （“WINDOWS”+“R”键，输入 inetcpl.cpl亦可）
点开高级往下拉，勾上”使用TLS 1.2”选项，或者点还原高级设置。
